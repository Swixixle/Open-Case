# Security

If you believe you’ve found a security vulnerability in OPEN CASE, please **do not** open a public issue.

**Preferred:** Use [GitHub private vulnerability reporting](https://github.com/Swixixle/Open-Case/security/advisories/new) if enabled for this repository.

**Otherwise:** Open a security advisory request with enough detail to reproduce, or contact the repository maintainers through an appropriate private channel.

We care especially about issues that could expose private individuals’ data, bypass signing or audit trails, or enable abuse of investigation endpoints at scale.
