For reporters

Using this report

This report is a lead-generation tool. It identifies public-record connections for further reporting. Suggested next steps:

  1. Pick 1–3 signals from the top leads section
  2. Pull the original FEC filing using the source link above
  3. Pull the Senate vote record from Senate.gov
  4. If an LDA filing is shown: search lda.senate.gov for the registrant's active filings
  5. Request comment from the official's office

This report does not constitute evidence of wrongdoing. All findings are drawn from public government records. The methodology is documented above.
